Course Description

This course introduces the essential technologies, testing concepts, and techniques that are associated with browser-based applications. The students will be introduced to a foundation of Web system architecture and technologies; they will get knowledge of Web-specific concepts, testing practices and methodologies, website design principles and practices. This course is intended for beginner or intermediate testers/developers who want to develop a working knowledge of Web software testing methodology.
